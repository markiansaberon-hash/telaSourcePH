# FabricHub PH — MVP Documentation

## What is FabricHub PH?

A B2B fabric sourcing service in the Philippines. Customers send their fabric list, we source everything from 30+ Divisoria suppliers, consolidate, quote, and deliver.

## How the Business Works

1. Customer submits a fabric list (photo or text) via website form
2. Our team sources fabrics from trusted Divisoria suppliers
3. We send a consolidated quotation
4. Customer confirms and pays
5. We deliver everything in one shipment

## Tech Stack (MVP)

| Component | Tool | Cost |
|-----------|------|------|
| Website | Wix (Free/Starter) | Free – P500/mo |
| Order Form | Google Forms or Wix Forms | Free |
| Backend | Google Sheets | Free |
| Automation | Zapier (Free tier) | Free |
| Communication | Viber / WhatsApp | Free |
| Payments | GCash / Bank Transfer | Free |

## Key Metrics to Track

- Orders per week
- Average order value
- Quotation-to-confirmation rate
- Average turnaround time (submission → delivery)
- Error rate (wrong fabric/quantity)
- Customer retention rate

## Quick Start

1. Set up Google Sheets using the template in `google-sheets-template.md`
2. Build Wix site following `website-structure.md`
3. Connect form to Sheets via Zapier or use embedded Google Form
4. Train team on workflow in `workflow.md`
5. Use templates from `message-templates.md` for customer communication
